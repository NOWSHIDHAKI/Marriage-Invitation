⚙️ Setup Instructions

1
Create Supabase Account
Visit https://supabase.com and create a free account. Then create a new project named "wedding-rsvp".
2
Set Up Database Table
In Supabase SQL Editor, run the setup script from SUPABASE_SETUP_GUIDE.md. This creates the rsvp_responses table.
3
Copy Your Credentials
From Supabase Settings → API, get:
• Project URL: https://your-project.supabase.co
• Anon Key: eyJ....... (long key starting with eyJ)
4
Update Configuration
Edit supabase-config.js and replace the placeholder values with your actual credentials:
const SUPABASE_URL = 'https://your-project.supabase.co';
const SUPABASE_KEY = 'eyJ......';
5
Test Your Setup
Open invitation.html in your browser, fill out the RSVP form, and submit. You should see a success message.
6
View Responses
Open admin-dashboard.html to see all RSVP responses, statistics, and to export the guest list as CSV.
